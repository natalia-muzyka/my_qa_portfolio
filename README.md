# Natalia Muzyka - Manual QA Middle - Portfolio
## About me and my current work
I am a QA engineer for QATestlab, an outsourcing company. I help our customers create successful, high-quality software by taking care of the testing process. My job is to create test documentation based on provided requirements, execute tests, match the actual software behavior with the expected behavior, and deliver testing results.
I ensure the product matches the customers' business aims and end users' needs.

I’m in charge of:
- testing new features, 
- executing smoke testing, 
- regression testing, 
- retesting fixed bugs.

Right now, I am involved in the healthcare project, which includes the admin part, user app, and user website. The main purpose of the project is to help people manage their mental health. The main functionality is developing flexible tests and collecting users' (patients') answers (data), which can be analyzed by admins (doctors) later.
